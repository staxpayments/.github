## 📦 What's in this org

Tools to integrate Stax across web, mobile, and server-side — supporting the Stax product family including **Stax Pay**, **Stax Connect**, **Stax Bill**, and **CardX by Stax**:

- **SDKs & client libraries** — iOS, Android, JavaScript/TypeScript, Python, and more
- **Sample apps & quickstarts** — runnable examples for common integration patterns
- **Open tooling** — small utilities we maintain for the developer community

Each repository has its own README with installation and usage details.

> ℹ️ **About these repositories** — the code here is **published from Stax's internal source of truth** as versioned, reviewed releases. What you see is production-ready, not work-in-progress. These are read-only, published mirrors, so **external pull requests and contributions aren't accepted** — but we'd still love your input. See below.

## 💬 Suggestions & requests

Have a feature idea, a suggested change, or run into something that isn't working as expected? **Your Stax account representative is the best place to start** — they'll make sure your request reaches the right engineering team and keep you posted on where it lands.

Not sure who your rep is? Reach out through [staxpayments.com](https://staxpayments.com) — real people, ready to help — and we'll connect you.

## 🔒 Security

Found a security vulnerability? **Please don't open a public issue.** Report it privately via the `SECURITY.md` in the affected repository, or contact us through [staxpayments.com](https://staxpayments.com), so we can resolve it responsibly.
